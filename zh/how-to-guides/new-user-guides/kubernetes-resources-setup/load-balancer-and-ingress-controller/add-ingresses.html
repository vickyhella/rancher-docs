添加 Ingress

你可以为工作负载添加 Ingress，从而提供负载均衡、SSL 终止和基于主机/路径的路由。在项目中使用 Ingress 时，你可以通过设置全局 DNS 条目来将 Ingress 主机名编程到外部 DNS。

  1. 在左上角，单击 ☰ > 集群管理。
  2. 转到要添加 Ingress 的集群，然后单击 Explore。
  3. 点击服务发现 > Ingresses。
  4. 单击创建。
  5. 从下拉列表中选择一个现有的命名空间。
  6. 输入 Ingress 的名称。
  7. 创建 Ingress 转发规则。有关配置规则的帮助，请参阅本节。如果你的任何 Ingress 规则处理加密端口的请求，请添加证书以加密/解密通信。
  8. 可选：点击添加规则来创建其他 Ingress 规则。例如，在创建 Ingress 规则以引导主机名请求后，你可能想创建一个默认后端来处理 404。
  9. 点击右下角的创建。

结果：已将 Ingress 添加到项目中。Ingress 会开始执行你的 Ingress 规则。
